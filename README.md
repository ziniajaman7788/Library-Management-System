## Description
This project is a simple console-based Library Management System developed using C.
It allows users to view available books, issue books, return books, and calculate fines for late returns.
The system uses structures and basic control statements.

## How to Run the Project
1. Open any C compiler (CodeBlocks / Dev C++ / GCC).
2. Copy the code from library.c.
3. Compile the program.
4. Run the executable file.
5. Follow the on-screen menu instructions.
